# Roadmap

## Phase 0: Prototype

- Static TypeScript dashboard.
- Local trust assessment model.
- Submission draft and architecture docs.
- Adapter boundaries for Ethic-Latex / ERH and ADM.

## Phase 1: Engine Integration

- Call ERH REST/gRPC scoring from the gateway.
- Ingest ADM safety signals from live telemetry.
- Store assessments as JSON artifacts.
- Add exportable public-agency reports.

## Phase 2: Open Data and Field Validation

- Add open-data provenance checks.
- Add data freshness and completeness scoring.
- Pilot with care navigation, disaster support, or education access.
- Publish anonymized validation summaries.

## Phase 3: Public-Service Operations

- Add role-based reviewer workflows.
- Add localization and accessibility testing.
- Add continuous monitoring for deployed AI services.
- Package deployable containers and reference infrastructure.
